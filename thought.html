在 src/config/文件夹下有一个defaultSetting.js 文件，是来做布局的默认设置的，里面开头就已经列出您的可选配置， 例如：/** * 项目默认配置项 * primaryColor - 默认主题色 * navTheme - sidebar theme ['dark', 'light'] 两种主题 * colorWeak - 色盲模式 * layout - 整体布局方式 ['sidemenu', 'topmenu'] 两种布局 * fixedHeader - 固定 Header : boolean * fixSiderbar - 固定左侧菜单栏 ： boolean * autoHideHeader - 向下滚动时，隐藏 Header : boolean * contentWidth - 内容区布局： 流式 | 固定 * * storageOptions: {} - Vue-ls 插件配置项 (localStorage/sessionStorage) * */ 这些是默认配置项。 启动项目，打开右边侧栏，可以选择配置项。
